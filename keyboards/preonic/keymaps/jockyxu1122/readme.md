Yida's keymap for Preonic Rev3
===

## Layout

- Typing Layer

```
,-----------------------------------------------------------------------------------.
|  Fn  |   1  |   2  |   3  |   4  |   5  |   6  |   7  |   8  |   9  |   0  |   +  |
|------+------+------+------+------+------+------+------+------+------+------+------|
| Tab  |   Q  |   W  |   E  |   R  |   T  |   Y  |   U  |   I  |   O  |   P  |   -  |
|------+------+------+------+------+-------------+------+------+------+------+------|
| Shift|   A  |   S  |   D  |   F  |   G  |   H  |   J  |   K  |   L  |   ;  | Enter|
|------+------+------+------+------+------|------+------+------+------+------+------|
| Caps |  Alt |   Z  |   X  |   C  |   V  |   B  |   N  |   M  |   ,  |   .  |   '  |
|------+------+------+------+------+------+------+------+------+------+------+------|
| Ctrl |  Cmd |   /  |  Esc |   space     |  backspace  |   [  |   ]  |   \  |   `  \
`-----------------------------------------------------------------------------------'
```

- Fnuctiona Layer

```
,-----------------------------------------------------------------------------------.
|  xD  |  F1  |  F2  |  F3  |  F4  |  F5  |  F6  |  F7  |  F8  |  F9  |  F10 | VolUp|
|------+------+------+------+------+------+------+------+------+------+------+------|
|      |  F11 |  F12 |      |      |      | Home | PgUp | PgDn |  End |      | VolDn|
|------+------+------+------+------+-------------+------+------+------+------+------|
|      |      |      |      |      |      | Left | Down |  Up  | Right|      |      |
|------+------+------+------+------+------|------+------+------+------+------+------|
|      |      |      |      |      |      |      |      | Mute | Prvs | Next | Pl/Ps|
|------+------+------+------+------+------+------+------+------+------+------+------|
|AdTggl|      |      | Raise|             |    Delete   |      |      |      |      \
`-----------------------------------------------------------------------------------'
```

```
,-----------------------------------------------------------------------------------.
| Sound|      |      |      |      |      |      |      |      |      |      |Faster|

|------+------+------+------+------+-------------+------+------+------+------+------|

`-----------------------------------------------------------------------------------'
```

---

## Compile and flash

1. `cd` to `qmk_firmware/`.
1. `make preonic:jockyxu1122` and a `.hex` file will be generated under `qmk_firmware/`.
1. Use QMK Toolbox to flash.
1. Remove the `.hex` file.
